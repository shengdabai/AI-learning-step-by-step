## English

> Most AI tutorials are either too academic (papers and math) or too shallow ("10 prompts to try"). This one sits in the middle — **structured enough to follow step by step, practical enough to use today.** Every page was written while actually using these tools, not theorizing about them.

This repository is a **curated, four-stage learning path** that walks you from your first AI coding tool to building agents to thinking like a solo AI founder. Everything is **bilingual (CN/EN)** and **ready to read** — open any `.html` file in your browser, or read the `.md` guides directly.

### Why this exists

I'm a Chinese-language teacher (6,000+ students) building AI + teaching tools in public. As I learned Claude Code, AI agents, and local models for my own work, I turned the messy notes into clean, interactive guides — so the next person doesn't have to start from zero. This is that trail.

### Who it's for

- **Developers** who want to master AI coding tools (Claude Code, Copilot, Cursor)
- **Non-technical founders** exploring what AI can actually do today
- **Solo builders** hunting for startup ideas and product maps
- **Anyone** who prefers structured, visual, hands-on learning over abstract theory
